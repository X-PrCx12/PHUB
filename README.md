## Clone this project

```bash
> git clone https://github.com/X-PrCx12/PHUB
```

## Install the dependencies:
Before running the below command, make sure you're in the project directory that
you've just cloned!!

```bash
> bash install.sh
```

### Usage
```bash
> npm start
```
